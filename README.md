# Projeto - EDA e Modelagem Supervisionada

## Estrutura

- `data/raw/`: dados originais (sem alteracoes)
- `data/processed/`: dados tratados para modelagem
- `notebooks/`: exploracao e testes iterativos
- `src/features/`: funcoes de limpeza e engenharia de atributos
- `src/models/`: treino e comparacao de modelos
- `src/evaluation/`: metricas e validacao
- `reports/figures/`: graficos finais
- `reports/tables/`: tabelas finais
- `configs/`: parametros e configuracoes
- `docs/`: guias e documentacao
